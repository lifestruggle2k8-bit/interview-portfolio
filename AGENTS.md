# Project Rules

## Goal
Build and improve a personal website for interview use first, then expand it into a stronger personal brand site.

## Current Priority
1. clarify business value fast
2. show measurable achievements before biography
3. make contact information obvious
4. keep the site static-first unless dynamic features are truly needed

## Writing Style
- Traditional Chinese first
- concise, evidence-based
- avoid vague claims without proof
- every skill or strength should be tied to a real example or result

## Content Order
- Hero with role positioning
- Quantified results
- Representative cases
- About
- Contact

## Quality Checks
Before calling the site complete:
- verify there are no broken links
- verify mobile layout still reads clearly
- verify primary CTA is visible above the fold on desktop and mobile

## Tech Preference
- prefer plain HTML/CSS/JS for fast iteration
- only add frameworks when content volume or feature needs justify it
